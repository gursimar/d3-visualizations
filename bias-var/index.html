Discussion

What is polynomail regression?

Linear regression is a technique for modeling a dependent variable (y) as a linear combination of one or more independent variables x_i(X). Polynomial regression allows us to capture non-linear relationships between X and y using a change of basis (X=[1 X X^2 x^3 .. X^p]). In a nutshell, instead of a line, it allows us to fit a nth degree polynomail to the data. Read more [link].

What is the aim of this playground?

Tiner with different values of parameters p, noise, amount of sampling. Each values of the parameters result in a different model in the bias-variance tradeoff landscape. The aim is to demonstrate the fundamental tradeoff, relate it with the concept of overfitting/ underfitting and generalization error.

What is bias and variance? How do you compute it?

Imagine, we can rebuild different model by repeatedly sampling new data. Each time we sample the data, we may end up with a different model due to noise. The bias corresponds to the expected value of (say a particular prediction corresponding to a particular data point in the sample) and its true value. The variance corresponds to the spread of these predictions. Read more [link]. We compute the bias and variance of the model by taking average of bias and variance of all data point in the sample.

What is bias-variance tradeoff?

The tradeoff is in the sense that when you tweak the model parameters (the value of degree of degree of polynomial in this case), decrease in one type of error leads to increase in the other and visa verse.

Scatter plot; how to interpret it?

The scatter plot represents a single snapshot of the data (show in blue dots). We fit a nth degree polynomail regression line to it (shown in pink).

You can observe the bias variance tradeoff by clicking in the dataset icon repeatedly and observing that you get different model each time depending on noise. When this change is more, you have a model with high variance. On the other hand, if you choose a low degree polynomial which is a bad fit, you always get the same model. This means variance is low, however the bias is quite high.

Notice this sample can in itself be biased in the sense that it only represents a small part of your distribution. This is happening the case of dataset 3. This can be observed using the zoom-out feature.

Box plot; how to interpret it?

Box plot is generated by using different values of residuals (absolute difference between predictions of the model and its original value). Each bar corresponds to a different degree of polynomail. Notice that at the bar goes down, it suggets that errors are less (on average). Hence the bias of the model is low. Similarly, the height of the bar gives an idea of the spread of these residuals (and hence individual predictions), giving us an estimate of variance. These two things roughly corresponds to bias and variance of the model. Notice that as the model degree increases, the bars goes down (low bias) and also the height increases (high variance).

Bar plot; how to interpret it?

Bar chart is generated by computing the bias and variance of the model. Each bar corresponds to bias and variance of the model for a particular degree of polynomial. Notice, that as we go right (the model degree increases), the bias of the model decreases and the variance increases. On the other hand, if we go left (model degree decreases), the bias of the model increases and the variance decreases.

But, why did not you plot the bull's eye diagram?

Bulls's eye diagram (proposed here [link]) though intuitive, is misleading and not very accurate. The bias and variance is a one dimentional quantity and plotting on 2D space causes confusion in the mind of the learner. Also, the model is inaccurate since the spread and distance from the bulls eye does not correspond to true values/ or even scaled values. Read our project report for more details [link (coming soon)].
